# 数据模型

数据模型必须保存这个核心实验：

> 真人异常源进入 AI 群像，并改变叙事。

不要只存原始聊天消息。每条关键记录都应该携带控制者、可见性、场景和扰动上下文。

## Session

```json
{
  "id": "session_001",
  "title": "Royal Banquet Disturbance",
  "createdAt": "2026-05-27T12:00:00.000Z",
  "updatedAt": "2026-05-27T12:30:00.000Z",
  "mode": "immersion",
  "activeSceneId": "scene_001",
  "scenarioPreset": "web_novel",
  "room": {
    "worldview": "谣言可以成为法律的宫廷。",
    "background": "国王可能已经死亡。",
    "roleSlots": "公主、宰相、外邦使节",
    "aiWorldRules": "AI 维持连续性，人类制造剧情裂缝。"
  }
}
```

`scenarioPreset` 控制创作者整理和导出重点。支持值为 `web_novel`、`murder_mystery` 和 `virtual_theater`。

`room` 是内部字段名，对应界面里的 `创作背景卡`。它记录世界观、剧情背景、可客串角色槽位和 AI 世界维护规则，用于导出和脑暴；它不是在线房间，也不表示多人联机。

## Continuity Handoff

当控制权从真人回到 AI 时创建。它是一个“剧情连续性桥接”，用于让 AI 顺着被真人改变过的故事状态继续走。

```json
{
  "id": "handoff_001",
  "sessionId": "session_001",
  "sceneId": "scene_001",
  "intrusionId": "intrusion_001",
  "characterId": "char_eileen",
  "characterName": "Eileen",
  "visibility": "anonymous",
  "intrusionKinds": ["clue_contamination"],
  "awareness": "none",
  "awarenessScope": "controlled",
  "summary": "Eileen returned to AI control after 1 canonical human-controlled line(s) and 1 AI reaction(s).",
  "prompt": "[VistrTavern Continuity Handoff]\\nThese events are canonical...",
  "relatedMessageIds": ["msg_001", "msg_002"],
  "createdAt": "2026-05-27T12:10:03.000Z"
}
```

Awareness 模式：

- `none`：角色在世界内不知道真人控制这件事。
- `subtle`：角色可以感到迟疑、断片或失控感。
- `explicit`：角色可以把外部意志识别为世界内体验。

Awareness scope：

- `controlled`：只有恢复控制的角色表现异常意识。
- `observers`：旁观 AI 角色可以察觉异常。
- `both`：恢复角色和旁观者都可以表现异常意识。

Awareness 事件：

- `self_anomaly_awareness`：恢复控制的角色意识到自己的话或行为不对劲。
- `observer_anomaly_awareness`：旁观 AI 角色察觉恢复角色行为异常。

Intrusion kind 取值：

- `character_takeover`
- `anomaly_line`
- `memory_fracture`
- `external_will`
- `plot_hook`
- `relationship_sabotage`
- `clue_contamination`
- `world_rule_break`

## Branch Point

Branch point 用于标记真人 intrusion 打开的创作者剧情路线。它不会创建聊天消息。

```json
{
  "id": "branch_001",
  "sessionId": "session_001",
  "sceneId": "scene_001",
  "intrusionId": "intrusion_001",
  "characterId": "char_eileen",
  "characterName": "Eileen",
  "type": "identity",
  "title": "Forbidden bloodline reveal",
  "summary": "Eileen may know royal secrets that should be impossible for her to know.",
  "options": ["Expose the bloodline", "Hide the memory gap", "Let the chancellor weaponize the secret"],
  "createdAt": "2026-05-27T12:11:00.000Z"
}
```

## 导出契约

每次导出都应该包含：

- 会话总结。
- 场景列表。
- 入侵时间线。
- AI 接管连续性 handoff。
- AI 异常察觉事件。
- 剧情分支标记。
- 真人意图。
- 互动灵感捕获。
- 创作者脑暴笔记。
- Creator Pack Markdown，用于整理可复用写作素材、冲突钩子、分支路线和 continuity handoff 上下文。
- Organized Material Markdown，用于按场景类型整理异常发言、AI 反应、冲突钩子、分支路线、异常察觉素材和下一步可写方向。
- 高张力对话。
- AI 对入侵的反应。
- 人物关系变化。
- 世界状态变化。
- 必要时附带原始 JSON。
