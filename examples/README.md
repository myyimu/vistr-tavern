# VistrTavern Examples

These examples show what VistrTavern is trying to produce for creators: not a full transcript, but reusable dramatic material created by a human anomaly entering an AI ensemble.

Current examples are written in Chinese because the first public positioning is creator-facing Chinese documentation:

- [网文剧情种子](zh-CN/web-novel-scene.md)
- [AI 剧本杀线索污染](zh-CN/ai-murder-mystery.md)
- [虚拟剧场现实怀疑](zh-CN/virtual-theater.md)

Each example follows the same shape:

1. Scenario.
2. Before human intrusion.
3. Human anomaly line.
4. AI reaction.
5. Continuity handoff.
6. Creator-ready output.
